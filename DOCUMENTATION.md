# Расположение
Общие директории
```
Common - переработка механик самой террарии
Content - новые механики
UI - user interface
Localization - локализация
```

Поддиректории
```
Content/Buffs - кастомные (де)баффы
Content/Needs - реализация потребностей, т.е. голод, жажда...
Content/Skills - различные навыки, например ковка

UI/NeedBars - интерфейс для потребностей

Common/Commands - команды
Common/GlobalNPCs - все что связано с NPC
```

## Важно
HydrogenPlayer - сердце мода, которое обрабатывает действия игрока и хранит различные переменные связанные с ним
